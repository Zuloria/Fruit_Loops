
## Starta projektet/spelet

För att starta spelet skriver man följande i terminalen, medan man står i projektets rotmapp.

```commandline
python -m src.game
```

- Spelkaraktärens symbol som man styr = ♠
- Frukt/grönsaker man ska plocka upp = o
- Väggar som man inte kan passera = ■

För varje steg man tar så förlorar man poäng. Försök plocka upp alla föremål på
banan på ett genomtänkt sätt för att minska poängbortfall.

Man använder följande tangenter för att styra spelet:
- W = uppåt
- A = vänster
- S = nedåt
- D = höger

- I = inventory (se vad för objekt man har plockat upp)

- Tryck en tangent och sedan enter för att genomföra kommandot (ex, w och enter för att gå uppåt)

- När man inte vill spela längre trycker man antingen X eller Q samt enter för att avsluta spelet
